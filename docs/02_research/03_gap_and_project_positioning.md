# 6. Gap and Project Positioning

## Gap

Many public walkthroughs either:

- show only a simple baseline without modern transformer training, or
- show only transformer training without baseline comparison and careful error analysis.

## Positioning of This Project

This project fills that gap by providing:

- both baseline and modern model tracks
- reproducible training/evaluation steps in one notebook
- practical debugging notes for dependency and runtime compatibility issues

## What Is Novel Here

The novelty is not proposing a new model architecture. The value is a full, practical, reproducible implementation and analysis workflow for shallow parsing in a real development environment.
