# 15. Conclusions

## Summary

This project demonstrates a complete shallow parsing workflow from classical baseline to modern transformer training on CoNLL-2000.

## Main Findings

- A POS-feature logistic baseline is useful but limited on minority classes.
- Transformer fine-tuning yields very strong chunk-level F1.
- Practical reproducibility requires handling environment and API compatibility issues explicitly.

## Outcome

The project is now documented as a reproducible, structured case study suitable for submission, presentation, and extension.
